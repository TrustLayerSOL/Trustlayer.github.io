# TrustLayer Landing Page

Transparent reward layer for meme coins on Solana.

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` to see your site.

### Build

```bash
npm run build
```

This generates a production-ready build in the `dist` folder.

## Deployment

To deploy to GitHub Pages:

1. Update `vite.config.js` base if your repo is not at the root domain
2. Run `npm run build`
3. Push the `dist` folder or set up GitHub Actions for auto-deploy

## Tech Stack

- React 18
- Tailwind CSS
- Vite
- Lucide Icons
